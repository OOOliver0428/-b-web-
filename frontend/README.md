# B站房管工具 - 前端

简洁高效的B站房管工具前端界面。

## 功能特性

- 🎥 **实时弹幕显示** - WebSocket实时接收并展示弹幕
- 🔨 **快速禁言** - 一键禁言，支持多种时长选择
- 📝 **禁言管理** - 查看和解除禁言列表
- 🛡️ **敏感词过滤** - 自动检测并处理违规弹幕
- 👥 **用户管理** - 查看在线用户列表

## 快速启动

### 方式一：直接打开（推荐）

由于使用 CDN 引入依赖，直接用浏览器打开 `index.html` 即可：

```bash
# 进入前端目录
cd frontend

# 用浏览器打开（或者双击 index.html）
start index.html
```

### 方式二：使用本地服务器

如果需要手机访问或解决跨域问题：

```bash
cd frontend

# 使用 Python 启动简单服务器
python -m http.server 3000

# 或使用 Node.js 的 npx
npx serve -s . -l 3000
```

然后访问 http://localhost:3000

## 使用说明

### 1. 配置后端地址

点击右上角设置按钮，配置后端API地址（默认 http://localhost:8000）

### 2. 启动监听

1. 在顶部输入框输入直播间房间号
2. 点击"开始监听"按钮
3. 连接成功后即可看到实时弹幕

### 3. 禁言用户

**方式一**：鼠标悬停在弹幕上，点击"禁言"按钮

**方式二**：点击用户名打开菜单，选择禁言

**方式三**：在右侧"在线用户"列表中点击禁言

### 4. 敏感词管理

在右侧"敏感词过滤"面板中：
- 点击"添加"增加新的敏感词
- 点击敏感词的"×"删除
- 勾选"启用自动审核"自动处理违规弹幕

### 5. 解禁用户

在"禁言列表"面板中可以看到被禁言的用户，点击"解禁"按钮即可解除。

## 技术栈

- Vue 3 (CDN)
- Element Plus (CDN)
- Axios (CDN)
- Day.js (CDN)

## 注意事项

1. 确保后端服务已启动（python run.py）
2. 首次使用需要在 .env 中配置 B站 Cookie
3. 你的账号需要是直播间的房管才能禁言用户
4. 建议使用 Chrome/Edge 浏览器获得最佳体验
